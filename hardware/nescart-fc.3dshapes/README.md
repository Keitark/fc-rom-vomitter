# Project-local SRAM 3D model

`SSOP-32_11.305x20.495mm_P1.27mm.step` is used by the two
CY62128EV30LL-45SXIT SRAM footprints (U13 and U14). It is kept project-local because
the KiCad 10 footprint installed on the development machine referenced a
nonexistent `SOP-32...step` filename.
